html lang="en - Temos o costume de indicar o atributo lang no elemento html logo no início do documento para indicar que todo o conteúdo contido dentro do html está escrito em um determinado idioma. DOCTYPE html - A declaração de tipo de documento no HTML, também conhecida como DOCTYPE , é a primeira linha de código necessária em qualquer documento, HTML ou XHTML. A declaração de DOCTYPE é uma instrução para o navegador da web que diz em qual versão do HTML a página é escrita.
